lidarpy

Process, validate and inspect lidar measurements

lidarpy is a standalone Python package for lidar data processing. It includes Licel binary to NetCDF conversion, preprocessing, quicklooks, synthetic signal generation, and retrieval routines validated against synthetic truth.

Getting started Module references

Current package scope

  • Discover and convert Licel RAW measurements to NetCDF.
  • Apply core preprocessing corrections and quicklook generation.
  • Generate synthetic elastic, Raman and depolarization signals.
  • Validate retrieval algorithms with explicit physical assumptions.
  • Keep generic utilities separate from lidar-specific helpers.
